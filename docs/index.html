An opinionated microframework written on top of Amp with the goals of helping to create well-tested, cleanly-separated asynchronous PHP applications and reusable components we call "Plugins". If you're new to Labrador Kennel we recommend you start off with the tutorial Quick Start and continuing down the list of tutorials in order.

We expect you to have a solid understanding of how to write asynchronous PHP, particularly with Amp. If you are not familiar with async PHP we highly recommend you checkout the Amp Documentation before continuing.

Installation

Composer is the only supported method for installing this library. 

composer require cspray/labrador
