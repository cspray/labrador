Cspray\Labrador\Plugin\InjectorAwarePlugin::wireObjectGraph

Register an object graph onto the passed Injector.

You should refrain from calling Injector::make in this method and should ideally be defining parameters, sharing objects that are intended to be single instances, and configuring any factory methods onto your Injector. Making an object here is likely a sign that your implementation is doing too much.
