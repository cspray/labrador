Cspray\Labrador\Plugin\Pluggable::registerPlugin

Register a fully qualified class name that implements the Plugin interface that should be instantiated and loaded when loadPlugins is invoked.

If a Plugin is attempted to be registered AFTER loadPlugins is invoked an IllegalStateException SHOULD be thrown as all registration must happen prior to loading to simplify the process. If a Plugin is attempted to be registered that does not implement the Plugin interace an IllegalArgumentException MUST be thrown as all registered types must implement the minimum interface.
